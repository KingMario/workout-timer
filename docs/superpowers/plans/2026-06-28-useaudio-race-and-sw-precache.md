# useAudio Race and SW Precache Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix the async `NoSleep` lazy-load race and stop service worker install from precaching every built-in MP3.

**Architecture:** Move `NoSleep` import, instance, and generation state to one module-level singleton in `src/hooks/useAudio.ts`, while preserving the existing hook cleanup for timers, audio cancellation, and speech synthesis. Change the service worker generator so generated `public/sw.js` and `out/sw.js` only precache shell URLs, while runtime MP3 requests still use `cacheFirst`.

**Tech Stack:** Next.js 16, React 19, TypeScript, Vitest, Testing Library, npm, generated service worker script.

---

## File Structure

- Modify `src/hooks/useAudio.ts`: replace hook-local `noSleepRef` with module-level `noSleepModule`, add token-guarded enable/disable helpers, keep existing unmount cleanup and add module-level NoSleep disable there.
- Create `src/hooks/useAudio.test.ts`: isolated hook tests for the lazy-import race, happy path, cross-instance cancellation, and unmount cleanup.
- Modify `scripts/generate-service-worker.mjs`: remove the MP3 scan from `getPublicPrecacheUrls()` only; keep `getCacheVersion()` MP3 stamps and keep runtime `.mp3` matching.
- Regenerate `public/sw.js`: output from `node scripts/generate-service-worker.mjs`, committed because it is tracked.
- Do not commit `out/sw.js`: it is generated by `npm run build` and not tracked.

    await Promise.resolve();

    await Promise.resolve();

    mockNoSleepDisable.mockClear();

    second.api.disableNoSleep();

      await first.api.enableNoSleep();
    });
    expect(mockNoSleepEnable).toHaveBeenCalledTimes(1);

    first.unmount();
    expect(mockNoSleepDisable).toHaveBeenCalledTimes(1);

    const second = await renderUseAudio();
    await act(async () => {
      await second.api.enableNoSleep();
    });
    expect(mockNoSleepEnable).toHaveBeenCalledTimes(2);
  });

## Task 2: Implement Module-Level NoSleep State

**Files:**

- Modify: `src/hooks/useAudio.ts`

- [ ] **Step 1: Replace the hook-local NoSleep module state**

In `src/hooks/useAudio.ts`, replace the current module-level `noSleepImportPromise` and `loadNoSleep` block with:

```ts
type NoSleepLike = {
  enable: () => Promise<void> | void;
  disable: () => void;
};

const noSleepModule: {
  importPromise: Promise<typeof import('nosleep.js').default> | null;
  instance: NoSleepLike | null;
  generation: number;
} = {
  importPromise: null,
  instance: null,
  generation: 0,
};

const loadNoSleep = () => {
  if (!noSleepModule.importPromise) {
    noSleepModule.importPromise = import('nosleep.js').then(
      (mod) => mod.default,
    );
  }
  return noSleepModule.importPromise;
};

const enableNoSleepModule = async () => {
  if (typeof window === 'undefined') {
    return;
  }

  const generation = ++noSleepModule.generation;
  if (!noSleepModule.instance) {
    try {
      const NoSleepCtor = await loadNoSleep();
      if (generation !== noSleepModule.generation) {
        return;
      }
      noSleepModule.instance = new NoSleepCtor();
    } catch {
      return;
    }
  }

  if (generation !== noSleepModule.generation) {
    return;
  }

  try {
    await noSleepModule.instance.enable();
    if (generation !== noSleepModule.generation) {
      try {
        noSleepModule.instance.disable();
      } catch {}
    }
  } catch {}
};

const disableNoSleepModule = () => {
  noSleepModule.generation += 1;
  if (noSleepModule.instance) {
    try {
      noSleepModule.instance.disable();
    } catch {}
  }
};
```

Remove `const noSleepRef = useRef<NoSleepLike | null>(null);` from the hook body. Do not remove `audioCtxRef`, `speakTimeoutRef`, `dingTimeoutRef`, `recordedAudioRef`, `playbackIdRef`, or `currentAudioCancelRef`.

- [ ] **Step 3: Add beforeunload fallback without changing unmount behavior**

Add a separate `useEffect` inside `useAudio`:

```ts
useEffect(() => {
  if (typeof window === 'undefined') {
    return;
  }

  const handleBeforeUnload = () => {
    disableNoSleepModule();
  };

  window.addEventListener('beforeunload', handleBeforeUnload);
  return () => {
    window.removeEventListener('beforeunload', handleBeforeUnload);
  };
}, []);
```

This listener is a page-close fallback. It does not replace the normal hook unmount cleanup.
