# 《全身拉伸：缓解疼痛，高效放松》Workout Timer 计划整理

来源文件：`/Users/mario/Documents/books/全身拉伸：缓解疼痛，高效放松/全身拉伸：缓解疼痛，高效放松.md`

整理原则：

- 动态拉伸适合运动前或久坐后唤醒身体；静态拉伸适合运动后、睡前或放松恢复。
- 所有动作保持均匀呼吸，以“中等程度拉伸感”为准，不追求疼痛。
- 单侧动作在 `duration` 内左右交替或各半时间完成；需要更清楚的语音提示时，可后续拆成左右两个 step。
- 以下 JSON 均符合 `WorkoutPlan` 结构，可粘贴到自定义计划导入框。
